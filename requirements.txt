Explanation of Dependencies:
tkinter-html:

Provides the HTMLLabel widget for displaying HTML content in tkinter.
Install using: pip install tkinter-html.
folium:

Used to generate interactive maps.
Install using: pip install folium.
requests:

Handles API calls to fetch weather data.
Install using: pip install requests.
